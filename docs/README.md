## 기술 스택

### 핵심 라이브러리
- **clap**: CLI 인터페이스
- **tokio**: 비동기 런타임
- **notify**: 파일 시스템 감시
- **serde**: 데이터 직렬화
- **yara**: 위협 탐지 엔진

### 개발 도구
- **cargo**: 패키지 관리
- **rustfmt**: 코드 포맷팅
- **clippy**: 린팅
- **criterion**: 벤치마킹
